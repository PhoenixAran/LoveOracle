# Zelda Oracle

Fan made Zelda game.
Requires [LOVE 11.4](https://love2d.org/) and [Tiled 1.9](https://thorbjorn.itch.io/tiled).  
[love-imgui](https://github.com/MikuAuahDark/love-imgui) binaries will be required for some screens.
